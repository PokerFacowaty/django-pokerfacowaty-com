# pokerfacowaty.com

This is the Django-based pokerfacowaty.com website. This repository does not include the database holding most of the website's content.

## Dependecies:
- `Django >= 4.2`
- `paml2html >= 0.1.1`
- `python-dotenv >= 1.0.0`

All of these can be installed with Poetry or added manually.
